# voicebot
* An AI voicebot with added functions of RAG and memory!
* Ask questions from the context.
* Can fetch context from memory of chat history.

# Tech used
* Flask
* Groq
* Sqlite3
* FAISS
* SpeechRecognition
* SpeechSynthesis
